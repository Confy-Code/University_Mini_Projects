# University Mini Projects

Welcome to my repository archiving various academic mini-projects and practical implementations. Currently, this repository focuses heavily on core concepts within **Management Information Systems (MIS)** and business process optimization.

## Featured Focus: MIS & BPMN
The primary objective of the current projects is to map, analyze, and optimize organizational workflows using **BPMN (Business Process Model and Notation)**. 

* **Process Mapping:** Visualizing as-is and to-be business processes to improve organizational efficiency.
* **System Design:** Translating business requirements into structured technical workflows.
* **Documentation:** Analyzing case studies to design clear, compliant, and logical sequence flows, gateways, and pools.

## Other Academic Focus Areas
As this repository evolves, it will also house practical code and configurations from other domains:
* **Web Development & Design:** Crafting clean layouts with modern CSS frameworks and JavaScript.
* **Database Management:** Writing structured logic, procedures, and scripts for data systems.
* **Core Scripting:** Problem-solving and building foundational software components.

## Purpose
This repository serves as a centralized portfolio for my university coursework, demonstrating the bridge between business process analysis (MIS) and technical execution.
